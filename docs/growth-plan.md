# Growth Plan

## 1. Growth Thesis

Relay grows when private, high-signal interactions create social value that is amplified by transparent incentives.

Primary loops:
- Create -> engage -> earn
- DeadDrop scarcity -> unlock urgency -> return behavior
- Referral questing -> new user activation -> retained cohorts

## 2. North Star and Inputs

North Star:
- Weekly Active Participants (WAP): users who send or unlock at least one encrypted message and perform one social action

Input metrics:
- New key provisioning completion rate
- DM first reply time
- DeadDrop unlock rate
- Quest completion rate
- Referral-to-activation conversion

## 3. Acquisition Channels

- Farcaster channel launches and creator partnerships
- Time-limited DeadDrop campaigns
- Sponsored quests with ecosystem projects
- Referral multipliers for verified retained invites

## 4. Activation Strategy

Day 0 flow:
1. Sign in + key setup quest
2. Send first encrypted DM
3. Unlock first DeadDrop
4. Claim first FARM milestone

Target: 60%+ users complete all four steps in first session.

## 5. Retention Strategy

- Daily streak multipliers with quality thresholds
- Weekly themed DeadDrop events
- Inbox prioritization for trusted contacts
- Personalized “earn next” prompts based on behavior

## 6. Monetization Rollout

- Stage 1: minimal tip routing fee
- Stage 2: premium broadcast tiers
- Stage 3: sponsored quest marketplace + partner channels

## 7. Experiment Backlog

1. A/B: DeadDrop TTL (6h vs 24h)
2. A/B: first-contact friction (stake vs PoW)
3. A/B: referral reward timing (instant vs deferred)
4. A/B: streak reset grace window
5. A/B: quest ordering in onboarding
